# Ecommerce FAQ Chatbot (ML Based)

This project is an AI chatbot trained using Machine Learning and NLP
on an Ecommerce FAQ dataset.

Technologies:
- Python
- Flask
- Scikit-learn
- TF-IDF Vectorizer
- Logistic Regression

How to run:

1. pip install -r requirements.txt
2. python train_model.py
3. python app.py
4. Open https://ecommerce-chatbot-1-8d8f.onrender.com/
